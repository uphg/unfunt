# Funt

封装一些 JavaScript 常用的函数。

## 安装

使用 npm 安装

```sh
npm i funt
# or yarn add funt
```

在项目中导入

```js
// ES6 modules
import { each, find, ... } from 'funt'

// CommonJS modules
const { each, find, ... } = require('funt')
```

支持模块化按需引入

```js
// ES6 modules
import each from 'funt/each'

// CommonJS modules
const each = require('funt/each')
```
