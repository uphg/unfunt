# Funt

Funt.js is a function library for JavaScript

## Installation

Using npm:

```sh
npm i funt
```

import in project

```js
// ES6 modules
import { each, find, ... } from 'funt'

// CommonJS modules
const { each, find, ... } = require('funt')
```

Import on demand

```js
// ES6 modules
import each from 'funt/each'

// CommonJS modules
const each = require('funt/each')
```
